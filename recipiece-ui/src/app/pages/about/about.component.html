All about Recipiece

User Agreement Open Source
Recipiece is built on Angular and Python, and is available on Github under the GNU Affero General Public License v3.0. Bugs may be filed in the issue tracker on Github. If you're particularly interested in contributing, feel free to make a pull request on the repo.

Parsing of third-party recipes is handled via the wonderful Recipe Scrapers library.

Formatting of ingredient amounts is provided by Fraction.js
